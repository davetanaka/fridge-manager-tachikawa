# 🤝 Contributing to 立川田中家冷蔵庫管理

このプロジェクトへの貢献を歓迎します！

## 📋 貢献方法

### 🐛 バグ報告

バグを見つけた場合は、以下の情報を含めてIssueを作成してください：

- バグの詳細な説明
- 再現手順
- 期待される動作
- 実際の動作
- スクリーンショット（可能であれば）
- ブラウザ・OS情報

### 💡 機能提案

新機能のアイデアがある場合は、Issueで提案してください：

- 機能の概要
- 使用ケース
- 実装の提案（あれば）
- モックアップ（あれば）

### 🔧 プルリクエスト

1. このリポジトリをフォーク
2. 機能ブランチを作成 (`git checkout -b feature/amazing-feature`)
3. 変更をコミット (`git commit -m 'Add some amazing feature'`)
4. ブランチにプッシュ (`git push origin feature/amazing-feature`)
5. プルリクエストを作成

## 📝 コーディング規約

### TypeScript

- ESLintのルールに従う
- 型定義を必ず行う
- async/awaitを使用
- エラーハンドリングを適切に行う

### コミットメッセージ

```
feat: 新機能追加
fix: バグ修正
docs: ドキュメント更新
style: コードスタイル変更（機能に影響なし）
refactor: リファクタリング
test: テスト追加・修正
chore: ビルドプロセスやツールの変更
```

### ブランチ命名規則

- `feature/機能名` - 新機能
- `fix/バグ名` - バグ修正
- `docs/ドキュメント名` - ドキュメント更新
- `refactor/対象` - リファクタリング

## 🧪 テスト

プルリクエストを送る前に、以下を確認してください：

```bash
# ビルドが通ること
npm run build

# ローカルで動作確認
pm2 start ecosystem.config.cjs
curl http://localhost:3000/api/health

# エラーログを確認
pm2 logs --nostream
```
